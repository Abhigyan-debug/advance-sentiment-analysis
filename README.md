## Advanced Sentiment Analysis using Bi-LSTM

# About the Project

This project performs sentiment analysis on movie reviews using a Bidirectional LSTM (Bi-LSTM) model. The model predicts whether a review is positive or negative.

The IMDB Movie Review dataset is used for training and testing the model.

---

# Dataset

- IMDB Movie Reviews Dataset
- Total Reviews: 50,000
- Training Reviews: 25,000
- Testing Reviews: 25,000

---

# Libraries Used

- Python
- TensorFlow
- NumPy
- Matplotlib

---

# Model

The model contains:

- Embedding Layer
- Bidirectional LSTM Layer
- Dense Output Layer

---

# Steps Performed

- Loaded the IMDB dataset
- Performed basic Exploratory Data Analysis (EDA)
- Preprocessed the data using padding
- Built a Bi-LSTM model
- Trained the model
- Evaluated the model
- Predicted the sentiment of sample reviews

---

# Result

The model achieved around **85% test accuracy** on the IMDB dataset.
